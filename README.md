# Sortörés-tisztító

Egyszerű, helyben futó HTML-eszköz PDF-ből vagy weboldalról másolt szövegek tisztítására.

## Mit csinál?

- Az egyes sorvégi sortöréseket szóközzé alakítja.
- A dupla sortöréseket opcionálisan bekezdésként megőrzi.
- A `-↵` típusú elválasztásokat egybeírja.
- A kimenetet egy gombbal a vágólapra másolja.

## Használat

1. Nyissa meg a HTML-fájlt böngészőben.
2. Illessze be a szöveget a bemeneti mezőbe.
3. Kattintson a **Tisztítás** gombra.
4. Szükség esetén másolja ki a kimenetet.

## Korlátok

A program csak a dupla sortöréssel jelölt bekezdéseket őrzi meg.
Elsősorban szóközökkel tagolt, prózaszerű szövegekhez ajánlott. Versekhez, táblázatokhoz, kódhoz és nem szóközalapú írásrendszerekhez kevésbé alkalmas.
